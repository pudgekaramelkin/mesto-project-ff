
# **Проект Mesto**

## **О сайте**

Сайт Mesto - это интерактивная страница, на которой можно добавлять посты (карточки с фотографиями), удалять их и ставить карточкам лайк

## **Использованные технологии**

1. Java Script + Webpack

2. HTML

3. CSS
